This is a software meant to teach you how numbers work!
These are the commands it uses:
  add 'number' 
    - Adds the 'number' to the end of the list
  add 'number' 'position'
    - Adds the 'number' to the 'position' in the list
  delete index 'position'
    - Deletes the number at the 'position' in the list
  delete subsequence 'start_position' 'end_position'
    - Deletes the numbers starting from 'start_position' and ending at 'end_position'
  replace subsequence 'subsequence' with 'subsequence_rep' 
    - Replaces all the subsequences equal with 'subsequence' with 'subseuqnce_rep'
  display
    - Displays the sequence
  display menu
    - Displays this menu again
  display prime 'start_position' 'end_position'
    - Displays all prime numbers from the 'start_position' to the 'end_position'
  display even
    - Displays all even numbers from the 'start_position' to the 'end_position'  
  display sorted reverse
    - Displays the sequence sorted in a descending order
  sum subsequence 'start_position' 'end_position'
    - Displays the sum of all elements from 'start_position' to 'end_position' 
  gcd subsequence 'start_position' 'end_position'
    - Displays the greatest common divisor from 'start_position' to 'end_position'
  max subsequence 'start_position' 'end_position'
    - Displays the maximum number from 'start_position' to 'end_position'
  filter prime
    - Deletes all numbers that are not prime from the sequene
  filter negative
    - Deletes all numbers that are not negative from the sequence
  undo
    - Undoes the last action
  quit
    - Quits the program
